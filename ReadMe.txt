
Explanation
Classes:

User: Represents a user with name, gender, and age.
Driver: Represents a driver with details like name, vehicle, and location.
CabBookingApp: Manages users and drivers, handles ride searching and selection.
Distance Calculation: Uses the Euclidean distance formula to find drivers within a specified distance.

Demo Class: Contains the main method to demonstrate functionality by onboarding users and drivers, searching for rides, and choosing a ride.

Output
When you run the Demo class, it will simulate the scenarios and print the appropriate outputs.
